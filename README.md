 📋 项目简介

ExpenseTracker_77 是一个基于Web的个人支出管理应用，专门为期末考查项目开发。该项目帮助用户记录、统计和分析日常开支，提供完整的预算管理功能。

学号: 2024131077  
开发者:  郑康
项目类型: 期末考查项目  
技术栈: HTML5, CSS3, JavaScript, LocalStorage

🚀 功能特性

核心功能
- ✅ 支出记录管理** - 完整的增删改查功能
- ✅ 智能分类 - 餐饮、交通、购物、娱乐等6大类别
- ✅ 月度预算 - 预算设置与实时跟踪
- ✅ 数据统计 - 多维度支出统计分析
- ✅ 数据导出 - CSV格式数据导出

用户体验
- 📱 响应式设计 - 完美支持手机、平板、桌面设备
- 🎨 现代UI - 美观的界面设计和交互动效
- 💾 本地存储 - 数据安全存储在浏览器本地
- ⚡ 快速响应 - 流畅的操作体验
